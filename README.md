# Simple Todo App

## <a href='https://gulnara-yakupova-todoapp.herokuapp.com/'>Click to see the project</a>

You can:
  - see tasks
  - add new tasks
  - search tasks
  - mark tasks done
  - mark tasks important 
  - filter tasks active or done

## build
  - npm run build - to build
  - npm start - to start server (port 8080)
